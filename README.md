# RepRange

En Android-app för att beräkna rep range vid styrketräning.  
Byggd i **Kotlin** med **Jetpack Compose** och **MVVM**.

## Funktioner
- Mata in vikt och reps → appen räknar ut uppskattad 1RM.
- Visar prediktioner för högre vikter baserat på flera vanliga formler (Epley, Brzycki, O’Conner).


## Tech stack
- Kotlin
- Jetpack Compose
- MVVM
